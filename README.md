# CleanDataAssignment
Assignment for Coursework

You need these files in your working directory:

*   activity_labels.txt
*   features.txt 
* X_test.txt
* y_test.txt
* subject_test.txt
* X_train.txt
* y_train.txt
* subject_train.txt


You can then run **run_analysis.R** which will create a **tidy_data.txt** in your working directory.
